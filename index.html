Instalar a app

Instala a Leituras no teu dispositivo para acesso direto como uma app nativa.
📱 iPhone/iPad: Safari → Partilhar → "Adicionar ao ecrã principal"
🤖 Android: Chrome → Menu ⋮ → "Instalar app" ou "Adicionar ao ecrã inicial"
💻 Windows/Mac: Barra de endereço → ícone ⊕ Instalar
